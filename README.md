# Mastodon API

A selection of [Mastodon](https://docs.joinmastodon.org/) API methods implemented in Node.js.

## Requirements

- Node v20

## Install dependencies

```shell
$ npm install
```

## Run tests

```shell
$ npm test
```

## Create tarball

```shell
$ npm pack
```

## Install (in another module)

```shell
$ npm install mastodon-api-x.x.x.tgz
```
